# Hot Dog Stand
A program that uses a class named HotDogStand to simulate a network of hot dog stands and their common functions.

## Setup
1. Download and install [Git](https://git-scm.com/downloads), if you don't already have it.

2. Open terminal and run the git clone command:

   ```
   $ git clone https://github.com/cramaechi/hot-dog-stand.git
   ```
    or [download as ZIP](https://github.com/cramaechi/hot-dog-stand/archive/master.zip).

3. Go to the project folder:

   ```
   $ cd hot-dog-stand
   ```

4. Compile:

   ```
   $ make
   ```
   
## Usage
Run:

```
$ ./hotdogstand
```
